My Projects

Gazetteer

Made at IT Career Switch

The Gazetteer project is a mobile-first application that provides real-time information about users' surroundings using geolocation technology and REST APIs. It employs PHP routines for backend processing and cURL for API requests, integrating data dynamically with AJAX. The app features an interactive map built with Leaflet.js, displaying country borders and points of interest such as airports, parks, and museums. Users can access comprehensive country data, weather updates, regional news, and a built-in currency converter for real-time exchange rates, all within a responsive Bootstrap framework.

Company Directory

Made at IT Career Switch

The Company Directory project is a web-based application designed to manage company personnel, departments, and locations. It utilizes a MariaDB database to store and retrieve data, with PHP routines handling server-side logic. AJAX requests, facilitated by jQuery, ensure dynamic content updates without full page reloads. The interface is built using HTML and styled with Bootstrap for a responsive design, enabling users to easily add, edit, delete, and retrieve records within the directory (CRUD operations). This combination of technologies provides a seamless and efficient user experience.
